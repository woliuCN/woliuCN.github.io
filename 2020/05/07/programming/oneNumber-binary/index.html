二进制中1的个数

题目描述:

输入一个整数，输出该数二进制表示中1的个数。其中负数用补码表示。

思路:

方法一：先将整数转换为32为二进制数字，正数的话直接使用正则表达式匹配1的数字的结果，求其长度，负数的话需要先取其补码，这时候就用到了之前写的长整数相加了。最后也是用正则表达式匹配1的结果。

方法二：不断左移与数字进行与运算计算出1的个数,在与运算的时候负数会先取补码。
